  • EchoSynth

    November 2019

    A Music Information Retrieval(MIR) project to synthesize naturally occurring sounds into a standard midi file. This project uses a microphone to record naturally occurring sounds, then translates that sound into a set of sinusoids. The end user can then tweak the frequency, wave function, phase, amplitude and envelope to achieve the desired synth sound. Users are welcome to export the midi file and use the synthesized sound in a music production environment!

  • YOLO Real Fast

    August 2019

    A distributed system for annotating the context of a video "Real Fast". This system utilizes docker and Kubernetes to distribute processing tasks over a network of workers. It does this by first pre-processing a video file and splitting it at each context change(calculated by significant changes in the colour histogram). Each sub-clip is then sent over the Kubernetes network to individual workers, which compute the context of that clip using the "You Only Look Once" model. This model roughly identifies the content of that clip throughout the clip. Once all workers have completed their context annotation, their results are compiled into a single document to provide context annotation over the entire video file.

  • UVic Aero Onboard Computer

    March 2020

    Designed and developed the architecture for the primary computer placed on a student-designed uncrewed aerial vehicle(UAV). This software ran on a raspberry and was responsible for communicating with video and photo hardware, transmitting video feeds over a network, and performing light computer vision tasks. The system was utilized when the University of Victoria won the Unmanned Systems Canada student competition.
